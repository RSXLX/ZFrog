# ZetaFrog 项目概览

## 一、项目简介

ZetaFrog 是一个基于 ZetaChain 的桌面宠物 + Web3 应用项目，融合了可爱的青蛙 NFT、跨链旅行、社交互动等功能。

## 二、项目架构

```
ZetaFrog/
├── backend/           # Node.js 后端服务 (TypeScript)
├── frontend/          # Web 前端 (React/Next.js)
├── desktop_pet/       # Python 桌面宠物应用
├── contracts/         # 智能合约 (Solidity)
└── docs/              # 项目文档
```

## 三、技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | React + TypeScript | Web 应用 |
| 后端 | Node.js + TypeScript | API 服务 |
| 桌面端 | Python + PyQt | 桌面宠物 |
| 智能合约 | Solidity | ZetaChain 合约 |
| 数据库 | PostgreSQL | 持久化存储 |

## 四、核心模块

### 4.1 用户系统
- 钱包连接 (WalletConnect)
- 用户资料管理
- 身份验证

### 4.2 青蛙 NFT 系统
- NFT 铸造与管理
- 青蛙属性与状态
- 图片生成

### 4.3 旅行系统
- 跨链旅行
- 旅行日记生成
- 奖励分发

### 4.4 社交系统
- 好友系统
- 社区花园
- 互动功能

### 4.5 徽章与成就
- 徽章收集
- 成就系统
- 奖励机制
